Description

Given the item price (base cost of a item) and tax percent (the percentage of the item price being added as tax) for an item, find and print the item's total cost.

Input Format

There are 2 lines of numeric input:
The first line has a double, itemCost(the cost of the item before tax)
The second line has a integer, taxPercentage(the percentage of itemCost being addded as tax)

Output Format

Print the total item cost, where totalCost is a double with 2 decimal point.

Sample Input

12.00
8

Sample Output

12.96
